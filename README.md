# KarteikartenApp

# Deploy Site on Firebase
https://firebase.google.com/docs/hosting/deploying


## Requirements

* Install firebase tools
```
npm install -g firebase-tools
```
* Login to firebase account
```
firebase login
```

## Deploy
```
npm run build
firebase deploy
```
